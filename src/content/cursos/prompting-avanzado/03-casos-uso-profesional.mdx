# Casos de Uso Profesional

Aprende a aplicar técnicas de prompting en situaciones reales de trabajo.

## Redacción de Contenido

### Email Profesional
```
Escribe un email profesional para:
- Destinatario: Cliente que no ha pagado en 30 días
- Tono: Firme pero cordial
- Objetivo: Solicitar pago sin dañar la relación
- Longitud: Máximo 150 palabras
```

### Contenido para Redes Sociales
```
Crea 5 posts para LinkedIn sobre [tema]
- Audiencia: Profesionales de tecnología
- Estilo: Educativo con gancho inicial
- Incluir: Emoji relevante y call-to-action
```

## Análisis de Datos

```
Analiza estos datos de ventas y:
1. Identifica las 3 tendencias principales
2. Sugiere posibles causas
3. Recomienda acciones específicas

[Pegar datos aquí]
```

## Programación

```
Revisa este código y:
1. Identifica bugs potenciales
2. Sugiere mejoras de rendimiento
3. Verifica buenas prácticas de seguridad

[Código aquí]
```

## Investigación

```
Necesito entender [tema complejo].
Explícamelo en 3 niveles:
1. Resumen ejecutivo (2 oraciones)
2. Explicación intermedia (1 párrafo)
3. Detalle técnico con ejemplos
```
