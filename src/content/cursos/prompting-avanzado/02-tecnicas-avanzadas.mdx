# Técnicas Avanzadas de Prompting

Lleva tus habilidades de prompting al siguiente nivel con técnicas utilizadas por expertos.

## Chain of Thought (Cadena de Pensamiento)

Pide a la IA que "piense paso a paso" para problemas complejos.

```
Resuelve este problema paso a paso:
Un tren sale de Madrid a las 9:00 AM a 120 km/h...
```

## Few-Shot Learning

Proporciona ejemplos del formato o estilo que esperas:

```
Clasifica estos tweets como positivo, negativo o neutral:

Tweet: "¡Me encanta este producto!" → Positivo
Tweet: "Llegó tarde y roto" → Negativo
Tweet: "El paquete llegó hoy" → Neutral

Tweet: "Nunca volvería a comprarlo" →
```

## Role Prompting

Asigna un rol específico a la IA:

```
Actúa como un experto en SEO con 10 años de experiencia.
Analiza esta página web y dame 5 recomendaciones...
```

## Iteración y Refinamiento

No esperes la respuesta perfecta al primer intento. Refina tu prompt basándote en los resultados.
