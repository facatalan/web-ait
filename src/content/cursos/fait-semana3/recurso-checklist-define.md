# CHECKLIST DEFINE
## Los 5 Componentes de una Definición Efectiva

---

```
    ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
    │Intención │ │ Contexto │ │ Criterios│ │Restricc. │
    │Estratég. │ │Específico│ │ de Éxito │ │          │
    └──────────┘ └──────────┘ └──────────┘ └──────────┘
    ════════════════════════════════════════════════════
    ┌──────────────────────────────────────────────────┐
    │              CONTEXTO BASE                       │
    │         (Tu Cerebro Digital - Semana 2)          │
    └──────────────────────────────────────────────────┘
```

---

## BASE: CONTEXTO BASE (Tu Cerebro Digital)

- [ ] ¿Cargué mi Perfil de Claridad?
- [ ] ¿Cargué mi Contexto Base organizacional?

**Nota:** Este componente ya lo tienes construido desde la Semana 2. Solo lo cargas, no lo redefines cada vez.

**Pregunta clave:** *"¿Cargué mi Cerebro Digital?"*

---

## 1. INTENCIÓN ESTRATÉGICA

- [ ] ¿Qué quiero lograr REALMENTE? (no la tarea, el objetivo detrás)
- [ ] ¿Qué decisión tomaré con esta información?
- [ ] ¿Qué cambiará cuando tenga el output?

**Pregunta clave:** *"¿Para qué necesito esto?"*

---

## 2. CONTEXTO ESPECÍFICO

- [ ] ¿Qué información particular de ESTE problema necesita la IA?
- [ ] ¿Cuáles son los antecedentes de esta situación?
- [ ] ¿Por qué importa ahora? ¿Qué ha pasado recientemente?
- [ ] ¿Hay datos específicos que deba incluir?

**Pregunta clave:** *"¿Qué es único de este caso?"*

---

## 3. CRITERIOS DE ÉXITO

- [ ] ¿Cómo sabré que el output está bien?
- [ ] ¿Qué características debe tener el resultado?
- [ ] ¿Qué nivel de detalle necesito?
- [ ] ¿Qué formato funciona mejor para mi uso?

**Pregunta clave:** *"¿Cómo se ve un buen resultado?"*

---

## 4. RESTRICCIONES

- [ ] ¿Qué NO quiero?
- [ ] ¿Qué límites debo respetar? (presupuesto, tiempo, recursos)
- [ ] ¿Qué excluir explícitamente?
- [ ] ¿Hay preferencias de estilo o formato que mencionar?

**Pregunta clave:** *"¿Qué evitar o excluir?"*

---

## EJEMPLO RÁPIDO

**Sin estructura:**
> "Necesito entender si debemos expandirnos a Perú"

**Con los 5 componentes:**

> **[CONTEXTO BASE - ya cargado]**
> "Soy CEO de consultora chilena de 50 personas, $5M facturación, especializada en transformación digital."
>
> **[CONTEXTO ESPECÍFICO]**
> "Evalúo abrir oficina en Lima porque perdimos un cliente clave y queremos diversificar geografía."
>
> **[INTENCIÓN]**
> "Necesito decidir si vale la pena y cuánto invertir."
>
> **[CRITERIOS]**
> "Dame: tamaño de mercado, top 5 competidores, barreras regulatorias, 3 escenarios de inversión con ROI."
>
> **[RESTRICCIONES]**
> "Inversión máxima año 1: $200K, no puedo relocalizar más de 2 socios."

---

## RECORDATORIO

> El **Contexto Base** ya lo tienes. Solo lo cargas.
>
> Los otros **4 componentes** son lo que defines para cada problema.
>
> **2 minutos definiendo = 20 minutos ahorrando en correcciones**
